## 🚀 What is this?

This is the central hub of the PrimeOps ecosystem.

A practical operational environment designed to **build, test and document real-world infrastructure and troubleshooting scenarios**.

It connects:

* 📂 Projects (labs, automation, infrastructure)
* 🧠 Method (PrimeOps operational framework)
* 🤖 HootAI (AI-assisted operations layer)
* 🏗️ Datacenter Lab (infrastructure simulation)

The objective is not only to showcase projects, but to **develop operational maturity through execution and documentation**.

* Real troubleshooting scenarios (network, infrastructure, services)

* Monitoring setups and alert validation
* Automation workflows (n8n)

## 🧩 System Overview

### 🟢 Projects

Hands-on implementations focused on:

* Observability (alerts → dashboards → runbooks)
* Automation (n8n workflows)
* Networking & troubleshooting labs
* Cloud security fundamentals

→ All projects are **artifact-driven** (docs, configs, runbooks)

---

### 🧠 PrimeOps Method

Operational framework designed to move from:

> reactive execution → structured engineering

Core structure:

* P → Purpose
* R → Rules
* I → Impacts
* M → Map
* E → Execution
* O → Observability
* PS → Standardization

Used to guide:

* troubleshooting
* incident response
* automation
* documentation

---

### 🤖 HootAI Layer

HootAI is an AI-assisted operational layer designed to support **structured troubleshooting and decision-making**.

It does not replace engineers — it **reinforces operational thinking** using:

* guided workflows
* runbooks
* incident templates

Positioning:

> AI-assisted operations, not AI replacement

---

### 🏗️ Datacenter Lab

A hybrid infrastructure lab used to simulate real-world environments and test operational scenarios.

Core components:

* Proxmox VE (virtualization)
* pfSense (network segmentation & firewall)
* Windows Server (AD, DNS, GPO)
* Linux (GitLab, Web, monitoring)
* SIEM/SOAR (in progress)

Focus:

* system behavior understanding
* incident simulation
* observability validation
* security fundamentals

> Some components are still under development.
